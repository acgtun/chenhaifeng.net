PerM2 is a software package for mapping DNA read sequence to genome reference sequence. The previous PerM is here.

Source Code and Compilation

Download the latest version source package from PerM2SourceCode.zip. Type the following command to unzip it.
$ unzip PerM2SourceCode.zip
Enter the source code folder and type the following command to compile the source code.
$ make
After compiling the source code, the executable file named aligner will be in the source code folder.

Quick Start

To run the command line, type aligner with the arguments in the order. 
$ ./aligner <reference> <reads> [options]
The reference and reads file should be fasta file. The suffix of the file can be fa, fasta or other. However, if the suffix of the reference file is ".sdindex", then the reference should be an index file generated by PerM2, as the following command shown. 
$ ./aligner hg19.sdindex reads.fa [options]
option arguments:
-v Set the maximum number of error allowed in the reads, the default value is 2.
-o Set the output file name, the defalut value is the same as the read file name.
-s Save the reference index, the index file name is the same as the reference file name, except the suffix. 
$ ./aligner hg19.fasta reads.fa -v 3 -o out.sam -s
